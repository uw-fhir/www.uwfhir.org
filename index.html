Goal

Our goal is to bring together individuals from across many disciplines to learn about the healthcare data standard FHIR (Fast Healthcare Interoperability Resources). Attendees will leave with an understanding of how FHIR has been used to drive and manage healthcare innovations at other institutions. This 1.5 day conference will feature a half-day technical tutorial where participants will learn how the data standard FHIR is structured, what tools are available to develop applications that use FHIR, and how to develop applications for clinical use. The rest of the conference will feature panels and talks aimed at de-mystify the FHIR standard and contextualizing how FHIR can enable healthcare organizations to manage their technological innovation.

What is FHIR and why should I care?

Fast Healthcare Interoperability Resources (FHIR) is a data standard for healthcare data developed by Health Level Seven International (HL7). Healthcare data that is stored using different formats and standards is a major problem facing healthcare organizations. The ability to share data between different organizations is limited; further, applications that work in one healthcare systems electronic health record (EHR) may not work in other organizations EHR because of the differing data structures and standards.

FHIR is proving to be the best attempt at developing a unify data standard that will allow interoperability between institutions for data sharing. The end goal is to enable the "EHR as a platform" concept, much like the apple or android app stores, where any application developed to use FHIR data can be downloaded and used on any EHR that has adopted FHIR.
